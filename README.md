# PGO-2
Repozytorium zawierające program w Javie symulujący mini bibliotekę.
Klasy:
- Main - symulacja
- Ksiazka - dane książki (tytuł, autor, liczba stron, dostępność), wypis informacji, wypożyczenie i zwrot danej książki
- Czytelnik - dane czytelnika oraz licznik wypożyczeń
- Biblioteka - tablica książek (dodawanie, wypisywanie, wyszukiwanie, liczenie dostępnych, wypożyczenie i zwrot przy użyciu tytułu oraz czytelnika)
